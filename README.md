# nn_ploter
A Python demo for plotting neural networks.

## Requirements
- Environment: python3
- Lib: [Matplotlib](https://matplotlib.org/)

#### install matplotlib
```
pip install matplotlib
```

## Usage
Run
```
demo.py
```
